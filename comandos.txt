1. Cuando iniciamos un proyecto nuevo ejecutamos:
git init
2. Agregar cambios al area de preparacion
git add .
3. confirmar los archivos hechos y los cambios
git status
4. versionar el proyecto local
git commit -m "mensaje descriptivo"
5.crear nuevo repositorio en github y seguir intrucciones de comando
6.verificar si quedo bien enlazado:
git remote -v
7.subir los cambios locales al repositorio remoto (en la nube)
git push -u origin main
